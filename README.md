# Calculadora PL – Tampermonkey

Script Tampermonkey para facilitar o cálculo da PL no simulador de preços do SF.

---

## Como instalar

1. Instale a extensão Tampermonkey;
2. Ative o modo desenvolvedor;
3. Clique no link abaixo para instalar o script:

https://github.com/L0R0Lucas/Calculadora-de-PL/raw/main/Calculadora-de-PL.user.js

3. Confirme a instalação no Tampermonkey

---

## Segurança

- O script roda apenas no navegador
- Não envia dados
- Não altera informações do sistema
- Apenas realiza cálculos locais

---

## Atualizações

Caso o simulador sofra alterações, o script poderá ser ajustado
e a atualização será distribuída automaticamente.
